## Installation
Add `@braks/revue-draggable` dependency using yarn or npm to your project

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn add @braks/revue-draggable
  ```

  </code-block>
  <code-block label="NPM">

  ```bash
  npm install @braks/revue-draggable
  ```

  </code-block>
</code-group>
